HTML

HTML son las siglas de HyperText Markup Language,que puede traducirse como lenguaje de marca o marcado de hipertexto

El lenguaje HTMLse emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro, y tambien puede cambiar de una vercion a otra de un mismo navegador, pero suelen tener un nombre similar.

HTML se compone de etiquetas, que se escriben entre los simbolos menor y mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creo la primera pagina web

Versiones de HTML

Tim Berners-Leedefinio la primera version de HTMLen el año 1991
